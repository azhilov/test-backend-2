Необходимо создать бекенд для сервиса укорачивателя ссылок используя:

Symfony (>=4.2)
MySQL
Docker

Фронтенд делать не нужно, достаточно:
+ API
+ документации
+ тестов

Оформите работу так, будто передаёте её коллеге-фронтендеру на сборку.

Функции, которыми должен обладать бекенд:

+ Сокращение новой ссылки
+ Отработка сокращенной ссылки (переход по ней)
+ Регистрация нового пользователя сервиса
+ Аутентификация пользователя сервиса
+ Просмотр (поиск) ссылок, сокращенных аутентифицированным пользователем
+ Изменение конкретной ссылки аутентифицированного пользователя
+ Удаление конкретной ссылки аутентифицированного пользователя

Необязательные “бонусные” задачи:
+ Добавить подсчёт статистики переходов по ссылкам и вывести её в кабинет пользователя (создать такие API-методы)
+ Добавить категоризацию добавляемых пользователем ссылок
